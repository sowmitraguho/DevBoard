Question-1: What are the different ways to select an element in the DOM?

There are several ways to select an element in the dom. Some of the ways are listed below:

  • By calling ID: let element = document.getElementById("elementId");
  • By class Name: let elements = document.getElementsByClassName("myClass");
  • By Tag Name: let elements = document.getElementsByTagName("p");
  • By Query Selector: let element = document.querySelector(".myClass");
    let elements = document.querySelectorAll(".myClass");

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML, innerText and textContent are explained below:

  1. innerHTML: it recieves and sets all the html tag, class, attributes with content inside an element.
  2. innerText: it recieves and sets the text content inside an element.
  3. textContent: it recieves and sets the text content inside an element with hidden content.

Question-3: What is event delegation in the DOM?

Event delegation is the way of adding an eventListener to multiple elements by calling their parents element.

Question-4: What is event bubbling in the DOM?

Event bubbling is the way of adding an eventListener to an elements and its parent elements too.

Question-5: How do you create, add, and remove elements using JavaScript?

Process of creating element: using document.createElement() function.
Process of adding element: using document.appendChild(), document.append() function.
Process of removing element: using document.removeChild(), document.remove() functions.
